Machine Learning

Machine Learning is a subset of Artificial Intelligence. Machine Learning algorithms are brittle; They are able to quickly adapt dynamically to more data and does not require human intervention to make changes to the models.

Machine learning algorithms are described as learning a target function (f) that best maps input variables (X) to an output variable (Y): Y = f(X).

Regression

Regression is a statistical method to predict an outcome variable by understanding the relationship between the target variable and independent variables. A curve or line is fit to the data points or variables and deviation of these data points from the curve is minimised. It helps the data scientists to predict the best set of variables for their model. The two common types of regression techniques are :-

  • Linear Regression
  • Logistic Regression

Decision Trees

Decision Trees is a supervised machine learning technique where the model maximizes the information gain and gives the best suited decision based on all possible outcomes. A simple example of decision tree is like a tree where branches signify the outcomes and leaf signify a particular class label- mostly, a Yes or a No to any outcome. Decision Trees can be used to predictions as well as classifications. Thus there can be regression trees and classification trees. These techniques work well even with missing data and findings can be easily communicated and easily explained. Some of the common decision trees are:-

  • Classification and Regression Trees
  • C4.0 and C5.0
  • Conditional decision trees

Bayesian Algorithms

Bayes Algorithms apply Bayes theorem on condition probabilities P(A|B) with the assumption that predictor variables do not depend on one another. It is one of the oldest methods in machine learning and is more suitable where datasets are large and used more in text based applications. It calculates probability of target variable belonging to a particular class using likelihood that this predictor belongs to the mentioned class and also knowing the individual probabilities of the class and the predictor. Some of popular Bayes algorithms are :-

  • Naïve Bayes
  • Gaussian Naïve Bayes
  • Bayesian Belief Network
  • Multinomial Naïve Bayes

Support Vector Machine

Support Vector Machine is a supervised classifier which is more robust and effective compared to other classifiers. This algorithm works by plotting a n-dimensional space called vector space and then defining the data points as support vectors. Through eigen vectors, SVM then tries to divide data points by drawing a hyperplane and perform classification. SVM tries to maximize distance between various classes which allows it to perform better generalization. It can also perform classification by transforming linear kernel to radial kernel using RBF. SVM works efficiently on large vector spaces and gives better resistance against data overfitting.

Cluster Analysis

Cluster Analysis refers to segregation of similar data points into clusters based on some criterias. These techniques classify data points based on probabilities or euclidian distances. Clustering algorithms can be used prior to supervised technique to further improve the accuracy of results. Some of the clustering techniques are :-

  • K-means clustering
  • Hierarchical clustering

Ensemble Algorithms

Ensembling is a technique of combining several weak learners and performing regression or mean on their outcomes to make overall predictions. This gives them an advantage of not over-fitting. However, these algorithms may suffer if there is too much bias or variance in data. Some of the common ensembling techniques are :-

  • Bagging
  • Boosting
  • Random Forests

Dimension Reduction Techniques

With datasets having too many features, it is not advisable to feed them directly to them machine learning algorithms as many of them might be irrelevant. Using dimension reduction techniques, we can reduce to specific 'intrinsic' dimensions. Some of the common dimension reduction techniques are:-

  • Principal Component Analysis
  • Linear discriminant analysis
